The sample app contains three additional parts beyond Amber itself:

  • The elements in index.html <body> tag.
  • amber-contrib-jquery module, providing jQuery and a wrapper over it.
  • amber-contrib-web module, providing Web package that allows for DOM manipulation and is inspired by Seaside.
  • domite module, providing Amber-native DOM wrapper.
  • silk module, providing Silk package that allows for DOM manipulation, is meant as alternative to Web and is inspired by stream metaphor and mochikit.

Additionally, module amber-compat-es2015 is included by default. This polyfills JS Promise for non-modern browsers / old node.js.
